# ESDoc Flow Type Plugin (PoC)
**This plugin is proof of concept.**

## Install
```bash
npm install esdoc-flow-type-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc-flow-type-plugin", "option": {"enable": true}}
  ]
}
```

- `enable` is default `true`
