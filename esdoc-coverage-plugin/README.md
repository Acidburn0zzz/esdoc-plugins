# ESDoc Coverage Plugin
## Install
```
npm install esdoc-coverage-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc-coverage-plugin", "option": {"enable": true}}
  ]
}
```
